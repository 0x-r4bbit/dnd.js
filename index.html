DnD.js

Simple HTML5 Drag and Drop

Intoduction

DnD.js is a minimalist JavaScript Framework for simple HTML5 Drag and Drop. It uses the native HTML5 Drag and Drop API, which is supported by all modern browsers. There's no jQuery or Plugin you need, to do some fancy Drag and Drop in your web application.

Why HTML5 Drag and Drop?

There are many reasons why to use Drag and Drop in a web application. With Drag and Drop you give your users the opportunity to import files and data in many different cases. This could be an upload app or a web player. But that's not all. Importing files is just one way to use Drag and Drop. When using Drag and Drop within the browser, you give your users a much better experience when it comes to sorting things and stuff like that.

What makes DnD.js so awesome?

Simplicity. DnD.js is a lightweight library. So what das lightweight in this case mean? The reason why DnD.js is such a small library, is, that does nothing more then adding the drag and drop eventlisteners to the DOM Elements you specify. You still have full control and access to everything you need.

Basic Usage

Using DnD.js is bloody easy. Just decide on which elements the drag and drop events should be attached, then specify the different callbacks. Here's an example: 


    dnd.on('.droparea', {
      'enter': function (element, event) {
        console.log('Entered drop area...')
      },
      'leave': function (element, event) {
        console.log('Left drop area...');
      },
      'drop': function (data, element, event) {
        console.log('Dropped!');
      }
    });
